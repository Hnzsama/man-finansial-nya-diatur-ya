# UI & UX Rules

Aplikasi ini digunakan setiap hari.

Prioritaskan:

- kecepatan input
- keyboard shortcuts
- mobile friendly
- dark mode

---

## Dashboard

Dashboard harus menjawab:

- ke mana uang pergi?
- berapa pengeluaran bulan ini?
- apakah budget terlampaui?
- apakah target tercapai?

---

## Form

Form transaksi:

- maksimal 10 detik untuk input
- validasi real-time
- shortcut keyboard

---

## Jangan

- terlalu banyak animasi
- modal bertingkat
- grafik yang tidak berguna
